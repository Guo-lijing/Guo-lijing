git bash常用命令的使用及实验

简介：
git bash的一些常用命令，
如：cd ; rm ; rm -r ; mkdir ; ls ; cp ; touch ; pwd等，
本次实验将会使用 tar 命令对文件进行解包

具体解释：
一、cd 进入目录

cd 是change directory的简写，改变目录的意思，就是切换到哪个目 录下，如 cd e:\fff
切换 E 盘下面的 fff 目录。当我们用 cd 进 入文件夹时,我们可以使用通配符*, cd f* ,如
果E盘下只有一个f开头的 文件夹,它就会进入到这个文件夹.

二、pwd 打印目录路径

pwd 全称print working directory，打印工作目录，它会显示我们当前所在的目录路径。

三、ls 展示目录文件

ls list，列出当前目录中的所有文件，还有 ll 命令，能够列出更为详细的内容。

四、mkdir 新建目录

mkdir make directory，新建一个目录，就是新建一个文件夹。

五、touch 新建文件

touch 新建一个文件，如 touch index.txt 就是在当前目录创建一个index.txt文件

六、rm ; rm -r 删除文件、文件夹

rm remove，删除一个文件，rm -r -r就是递归，先将文件夹里的文件删除，再将文件夹删除。

本次实验将会使用 tar 命令，将先下载的videoClips进行解包，然后删除包文件，并使用浏览器播放
解包出来的视频。
具体操作：

cd /d/guoLijing/video //进入包文件所在目录
tar -xf videoClips //对videoClips文件进行解包
rm videoClips //删除videoClips文件
ls videoMP4 //列出解包后的文件夹的文件

总结：

本次实验用的tar命令是对文件进行打包和解压，其参数 -c 是创建新的归档文件（打包）；
-x 是从归档文件提取文件（解压）； -f 文件名 是指定归档文件名。
